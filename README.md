### Installing

* `pkg update`
* `pkg upgrade`
* `pkg install git`
* `apt install python`
* `apt install python2`
* `apt install python3`
* `rm -rf ULTRA_DDOS`
* `git clone https://github.com/U7P4L-IN/ULTRA_DDOS`
* `pip3 install requests`


#### Finished you have successfully downloaded Ultra-DDos now to launch!

* `cd ULTRA_DDOS`
* `python2 main.py`
